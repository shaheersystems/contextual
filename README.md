# contextual

## Installation

```bash
bun install
```

## Development

Run the CLI directly from its TypeScript entrypoint:

```bash
bun run src/cli.ts
```

Or use the package script:

```bash
bun run dev
```

## Build

```bash
bun run build
```

The distributable CLI is built from `src/cli.ts` into `dist/cli.js`.
